# Release checklist - puedolav.ar

## Antes de Play Console

- [ ] Dominio definido.
- [ ] Política de privacidad pública.
- [ ] assetlinks.json publicado en /.well-known/assetlinks.json.
- [ ] Package name final decidido.
- [ ] TWA abre sin barra superior de Chrome.
- [ ] AAB firmado generado.
- [ ] Keystore resguardada fuera del repo.
- [ ] SHA256 guardado fuera del repo.
- [ ] Icono 512x512 preparado.
- [ ] Feature graphic 1024x500 preparado.
- [ ] Screenshots mobile preparados.
- [ ] Descripción corta preparada.
- [ ] Descripción completa preparada.
- [ ] Email de soporte disponible.
- [ ] Data Safety completado.
- [ ] Política de privacidad consistente con Data Safety.
